# Professional-grade-Banking-API


Let's create a secure, riche, scalable and feature-rich banking system that handles:

- user verification
- KYC verification
- real time transaction
- multi-currency support
- creation of virtual cards
- fraud detection
- automated PDF statement generation
- Deposits, withdrawals, inter-account transfers


Let's implement :

- 2 factor authentication via OTP
- rate limiting
- comprehensive logging
- JWT Auth via Cookies
- async task processing with celery
- reverse proxy with NGINX
- API Documentation
- email notifications
- SSL certificates

Let's use the stack:

- PostGresSQL
- Redis
- RabbitMQ
- Docker
- Django
- Django Rest Framework
